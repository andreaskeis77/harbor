# Harbor User Stories v0.1

## Core stories

- As a user, I want to create a new project and keep it separate from all others.
- As a user, I want to reopen a project later and quickly understand its current state.
- As a user, I want to see the current handbook and evolve it over time.
- As a user, I want to collect sources and evidence within one project context.
- As a user, I want to review what changed since my last pass.
- As a user, I want to archive finished projects and later reuse them as blueprints.
